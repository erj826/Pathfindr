# Pathfindr

A Google Chrome Dev Tools extension to help us visualize the DOM tree!

### Install dependencies

```
$ npm i
```

### Build

```
$ npm run build
```

### Unpack into your Chrome browser

Load the `./build` directory as an unpacked extension.
